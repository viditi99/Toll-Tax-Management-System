# Toll Tax Management System

A web-based **Toll Tax Management System** developed to simplify toll pass management and payment tracking.  
This system allows users to register, buy toll passes, generate QR codes for verification, and manage toll records efficiently.

## 🚀 Features

- User Registration and Login
- Buy Toll Pass
- QR Code Generation for Pass
- Pass Expiry Tracking
- View Pass Details
- Admin Dashboard
- Revenue Tracking
- User Toll History
- Automatic Pass Status Update (Active / Expired)

## ⚙️ Installation Steps

1. Install **WAMP Server**
2. Copy the project folder into:


C:\wamp64\www\


3. Start **Apache and MySQL** in WAMP.

4. Import the database in **phpMyAdmin**.

5. Open browser and run:


http://localhost/tolltaxproject
